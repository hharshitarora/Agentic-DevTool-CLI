# Agentic-DevTool-CLI

A Node.js CLI tool built with Commander.js designed for observability and secret management in agentic development workflows.

## Features

- **secret-init**: Set up local environment variables and secret management.
- **trace-stream**: Listen for and stream observability traces to Honeycomb.

## Installation

```bash
npm install
npm run build
npm link
```

## Usage

```bash
agentic-cli secret-init
agentic-cli trace-stream
```
